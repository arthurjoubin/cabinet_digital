# Contexte du site
Cabinet Digital est un site internet qui référence les meilleurs logiciels pour les experts-comptables et les PMEs.
Contenu du site :
- Une liste de logiciels, avec une fiche complète pour chaque logiciel
- Une liste d'actualités, avec une fiche complète pour chaque actualité
- Une liste d'articles, avec une fiche complète pour chaque article

# Technos
Site web : cabinetdigital.fr (mais je travaille sur le code pour le moment)
Technologies utilisées :
- Framework backend : Django
- Frontend : Templates Django HTML
- Framework CSS : TailwindCSS

# Charte graphique
- Couleurs principales : noir et blanc
- Couleur d'accentuation : bleu ciel très clair (TailwindCSS sky-50) utilisé en fond et pour certains boutons
- Design épuré et moderne avec de léger arrondi et des bordures fines
